# orangepi

It is important to note that these are the sources for the **__OrangePi Zero2__** NOT THE OrangePi Zero2W, as that board has a very different layout from the Zero2. The Zero2 is closer to the MQ-Quad than the Zero2W is in terms of hardware design, despite not looking like it at all.

## Sources

All DTBs and Kernels were extracted from the OrangePi Downloads hosted here:

https://drive.google.com/drive/folders/1Xk7b1jOMg-rftowFLExynLg0CyuQ7kCM

DTS's were decompiled from the DTB's present in these images
